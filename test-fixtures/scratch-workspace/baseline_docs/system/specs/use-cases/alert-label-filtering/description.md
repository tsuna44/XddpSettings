## 2. 目的・ゴール

オペレーターがアラート一覧画面でラベルを指定して絞り込み表示できるようにする。大量のアラートから特定ラベルのデバイスに関するものだけを確認したいという要求に対応する。

---

## 3. アクター

| アクター | 種別 | 説明 |
|---|---|---|
| Operator | 人（Operator） | アラート一覧画面でラベルを指定して絞り込み表示する利用者（UR-002 に明示記述あり） |

---

## 4. 前提条件

- アラートイベントにラベル情報が含まれていること（SP-004：device.alert.raised イベントへのラベル一覧付加が前提）
- アラート一覧画面が表示可能であること

---

## 5. 事後条件（成功時）

- 指定したラベルに合致するデバイスのアラートのみがアラート一覧に表示されること

---

## 6. 主フロー概要

1. オペレーターがアラート一覧画面でラベルを指定する
2. alert-dispatcher が指定ラベルでアラート一覧をフィルタする（SP-005）
3. 絞り込み結果がアラート一覧画面に表示される
