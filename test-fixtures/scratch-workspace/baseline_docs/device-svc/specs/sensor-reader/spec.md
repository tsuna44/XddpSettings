## 2. 機能概要

センサーから周期的に値を読み取り、閾値判定してアラート判定材料を作る。
センサー値を1秒間隔でポーリングし、閾値（THRESHOLD_HIGH）を超えた場合に alert-dispatcher へ通知する。

センサーデータ取得時には、device-registry から取得した紐づくラベル一覧を付加する（CRS SP-003）。

---

## 4. 処理フロー

1. 1秒間隔でセンサー値をポーリングする
2. device-registry から device_id に紐づくラベル一覧を取得し、読み取り結果に付加する（SP-003）
3. 閾値（THRESHOLD_HIGH）を超えた場合、alert-dispatcher へ通知する
