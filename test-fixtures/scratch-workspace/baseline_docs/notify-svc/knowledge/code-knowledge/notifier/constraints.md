## パフォーマンス・非機能特性

### [CK-001] 通知先一覧の戻り件数の互換性
**観察:** src/notifier.py:dispatch() は通知先一覧を固定リストで全件送信する実装。ラベルフィルタ導入時はAPIの戻り件数が変わるため呼び出し側の互換性確認が必要。
**示唆:** ラベルフィルタ導入後、戻り件数が変動する前提でクライアント側の処理を見直すこと。
**出典:** CR-2026-900（SPO Section 5.6 より）
