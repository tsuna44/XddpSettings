## 2. 機能概要

通知メッセージ・レポート本文のテンプレートに変数を埋め込んで文字列を生成する。CR-2026-900 のラベル機能には直接関与しない（対象 SP なし）。

---

## 3. 入出力

### 3.1 入力

| 項目 | 型 | 説明 | 制約 |
|---|---|---|---|
| テンプレート | str | テンプレート文字列 | |
| 変数 | dict | テンプレートに埋め込む変数 | |

### 3.2 出力

| 項目 | 型 | 説明 |
|---|---|---|
| 展開済み文字列 | str | 変数を埋め込んだ通知文・レポート文 |

---

## 4. 処理フロー

1. テンプレート文字列と変数を受け取る
2. 変数をテンプレートに埋め込んで文字列を生成する
