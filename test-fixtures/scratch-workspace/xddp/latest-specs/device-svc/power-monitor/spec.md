## 2. 機能概要

デバイスの電源・バッテリ状態を監視し、異常時に alert-dispatcher へ通知する。
バッテリ残量を定期的にチェックし、低下時に alert-dispatcher 経由でアラートを発行する。ラベル機能には直接関与しない。
