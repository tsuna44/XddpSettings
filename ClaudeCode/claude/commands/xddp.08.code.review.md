`.claude/skills/xddp-verify/SKILL.md` を読み込み、変更後のソースコードを検証する。

## 実行内容

1. `.claude/skills/xddp-verify/SKILL.md` を読み込む
2. 変更設計書の「変更後の確認項目」を収集する
3. 変更後のソースコードを読み込み、以下を静的確認する：
   - 全変更箇所にADD/MOD/DELタグが付与されているか
   - タグの仕様番号が変更設計書と一致しているか
   - 変更設計書通りに実装されているか
4. TM自動生成の準備として仕様番号×ファイルの対応を確認する
