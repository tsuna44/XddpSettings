# 共通用語集

> 全プロジェクト横断で使用するドメイン用語・略語を管理します。
> プロジェクト固有の用語は `docs/projects/{name}/glossary.md` に記載してください。

---

## 用語一覧

| 用語 | 読み | 定義 |
|---|---|---|
| XDDP | エックスディーディーピー | eXtreme Design-Driven Process。設計書駆動の変更管理プロセス |
| CR | シーアール | Change Request。変更要求 |
| CRS | シーアールエス | Change Requirements Specification。変更要求仕様書 |
| SPO | エスピーオー | Specout Output。既存仕様書調査結果 |
| CHD | シーエイチディー | Change Design document。変更設計書 |
| ANA | エーエヌエー | Analysis memo。要求分析メモ |
| DSN | ディーエスエヌ | Design memo。実装方式検討メモ |
| TSP | ティーエスピー | Test SPecification。テスト仕様書 |
| TM | ティーエム | Traceability Matrix。トレーサビリティマトリクス |

## 更新ルール

- 用語追加時は五十音順（またはアルファベット順）に挿入する
- 定義が変わった場合は理由をコミットメッセージに記載する
