# ADR-0005: 知識参照ガイドを初回のみ生成する理由

Status: Accepted
Date: 2026-06-21

## Context

`{DOCS}/AI_INDEX.md` の「知識参照ガイド」セクションを、CR 完了のたびに生成し直すか、
初回のみ生成するかを決める必要がある。

## Decision

「## 知識参照ガイド」セクションが存在しない場合のみ生成し、既存の場合はスキップする。

## Consequences

参照先パターンはディレクトリ構造定数であり、CR ごとの更新は不要である。毎回生成し直す方式に
比べ、無駄な書き込み・差分を避けられる。
